# Todoes

Todo application for personal use

## Project setup

```sh
composer install
```

Install & run Yarn
```sh
yarn; yarn encore dev
```

Create DB
```sh
symfony console d:d:c
```

Update scheme
```sh
symfony console d:s:u --force
```

Load fixtures
```sh
symfony console d:f:l
```

Run project
```sh
symfony serve
```
